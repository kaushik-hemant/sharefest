Sharefest Terms of service:

Sharefest provides a Peer-to-peer file sharing/distribution services and enables direct file transfer between 2 or more users. Direct transfer means there are no servers of which the file is being hosted on.

General:

Sharefest website provides its users an infrastructure which enables the user to send a file of any type to other people via sharing a unique link.

Sharefest do not store or host the shared files or content on a server or in the cloud - the file only exists on the originator and later on the computers whom the originator shared the file with.

Sharefest only stores meta data of the shared content.

Sharefest runs in only within the browser with negligible use of PC resources or upload bandwidth and is working only while the user is browsing in the website, when the user leaves the website so, he is no longer sharing the file or its resources

Sharefest Connects end user machines to use multiple sources of data. Looks for faster/nearer sources of a file.

Sharefest does not review or have any control over Shared Content.

“Room” (a unique url) is an entity where the file can be shared in. when entering a “Room” you will receive the shared content.
